# 🌸 Iris Flower Species Predictor

A machine learning web application that predicts iris flower species based on sepal and petal measurements.

## 🚀 Live Demo
[Add your Streamlit Cloud URL here after deployment]

## 📋 Features
- Interactive sliders for input measurements
- Real-time predictions
- Probability distribution visualization
- Beautiful and responsive UI
- High accuracy Random Forest model

## 🛠️ Tech Stack
- **Frontend**: Streamlit
- **ML Model**: Random Forest Classifier (Scikit-learn)
- **Language**: Python 3.9+
- **Deployment**: Streamlit Cloud

## 📊 Model Performance
- Accuracy: ~97-100%
- Trained on 150 samples from Iris dataset
- Test-train split: 80-20
